# Sistema de Entrega de Marmitas

Projeto para gerenciamento de entregas de marmitas.

## Funcionalidades

- Cadastro de clientes
- Gerenciamento de pedidos
- Controle de entregas
- Sistema de pagamento

## Estrutura do Projeto

```
entrega-de-marmitas/
├── src/
│   ├── controllers/
│   ├── models/
│   ├── views/
│   └── routes/
├── public/
├── tests/
└── docs/
```

## Instalação

```bash
npm install
```

## Uso

```bash
npm start
```
